# trunky - Basic truncation and string tools

[![build status](https://secure.travis-ci.org/serbytrunky.png)](http://travis-ci.org/serby/trunky)

## Installation

      npm install trunky

## Usage

'''js

var copy = '';// Load copy
var truncate = require('truncate').truncate;


truncate(copy, 10);

Will truncate and add an ellipsis

```
